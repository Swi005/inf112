###Requirements
* Up-to-date Maven and Java installs

###How to install
1. Open cmd and cd to the project
2. Run ``mvn compile``
3. Run ``mvn package``
4. Run ``java -jar target/RoboRally-1.0-SNAPSHOT.jar``
